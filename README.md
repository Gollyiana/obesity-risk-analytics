# A Predictive Analytics and Interactive Visual Framework for Obesity Risk Assessment Based on Behavioral Diagnostics

## 📌 Project Overview
This repository contains a data-driven web application developed using **Python** and **Streamlit** designed to estimate and analyze an individual's obesity risk tier. Moving beyond traditional BMI tracking, this framework diagnoses metabolic risk by assessing multi-dimensional behavioral, dietary, and physical lifestyle choices.

## 👥 Team Members & Roles
* **Member 1 (Data Engineer & Domain Analyst):** Data processing, cleaning, and descriptive UI layout.
* **Member 2 (EDA & Visualization Specialist):** Advanced diagnostic charts and visual insight panels.
* **Member 3 (Machine Learning Engineer):** Multi-class classification modeling and validation hub.
* **Member 4 (UI Integrator & Tech Writer):** Core predictive interface logic and documentation control.

## 🛠️ Project Architecture
1. **Data Layer:** Processing raw patient behavioral records (`ObesityDataSet_raw_and_data_sinthetic.csv`).
2. **Analytics Engine:** Data normalization, categorical label encoding, and Random Forest classification.
3. **Frontend Application:** Streamlit multi-tab user interface for interactive diagnostics.
